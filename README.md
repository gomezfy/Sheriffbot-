## Quick Start

### Desenvolvimento Local (Replit)

1. **Configure Secrets**
   - Set `DISCORD_TOKEN` in environment secrets
   - Set `CLIENT_ID` in environment secrets
   - Optional: `DISCORD_CLIENT_SECRET`, `SESSION_SECRET` for web features

2. **Run the Bot**
   ```bash
   npm start
   ```

3. **Deploy Commands**
   ```bash
   npm run deploy
   ```
