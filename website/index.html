🤠 About Sheriff Bot

Sheriff Bot is a Discord bot with Wild West theme that offers a complete experience of economy, mining, games and much more!

Built with Discord.js v14, the bot uses Canvas to generate custom images (profile cards, wanted posters) and supports 4 languages (PT-BR, EN-US, ES-ES, FR).

🔧 Technologies:

Node.js 18+ Discord.js v14 Canvas JSON Storage

✨ Highlights:

  • ✅ Dual currency system (Tokens + Coins)
  • ✅ Weight-based anti-exploit inventory
  • ✅ 47 custom emojis for profiles
  • ✅ Automatic Western image detection
  • ✅ Animated progress bars
  • ✅ Complete JSON welcome system
  • ✅ 9 configurable log types
